Codestral 2 is Mistral's code generation specialized model built specifically for high-precision fill-in-the-middle (FIM) completion. It helps developers write and interact with code through a shared instruction and completion API endpoint. Because it masters code and can also converse in a variety of languages, it can be used to design advanced AI applications for software developers.

Codestral 2 is well-suited for tasks such as:

  - Code generation
  - Fill-in-the-middle completion
  - Software development applications
