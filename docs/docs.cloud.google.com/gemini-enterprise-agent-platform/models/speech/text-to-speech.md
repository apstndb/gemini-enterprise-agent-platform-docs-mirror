This page shows you how to use Vertex AI Studio to convert text to speech.

To learn how to convert speech to text, see [Convert speech to text](https://docs.cloud.google.com/gemini-enterprise-agent-platform/models/speech/speech-to-text) .

## Convert text to speech

To convert text to speech, do the following:

1.  In the Agent Platform section of the Google Cloud console, go to the **Vertex AI Studio** page.

2.  Click **Generate speech** .

3.  Select the **Text-to-speech** tab.

4.  Configure the parameters as follows:
    
      - **Text** : Enter the text that you want to convert to speech.
      - **Voice** : Select a voice that you want the speech to be in.
      - **Speed** : Use the slider or text field to enter a value for the speed of the speech. A value of `1` represents normal speaking rate and a value of `4` represents four times the normal speaking rate.

5.  Click **Submit** .

6.  To listen to the generated speech, in **Speech** , click the play\_arrow **Play** button.

7.  To download the generated speech file in `.wav` format, click download **Download** .

## What's next

For more information on how to use Cloud Text-to-Speech (such as limitations, available voices, and how to set up Cloud TTS in your project), see the [Cloud Text-to-Speech documentation](https://docs.cloud.google.com/text-to-speech/docs) .
