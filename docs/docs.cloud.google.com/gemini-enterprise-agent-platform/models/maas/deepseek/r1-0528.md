To self-deploy the model, navigate to the [DeepSeek R1 (0528) model card](https://console.cloud.google.com/agent-platform/publishers/deepseek-ai/model-garden/deepseek-r1) in the Model Garden console and click **Deploy model** . For more information about deploying and using partner models, see [Deploy a partner model and make prediction requests](https://docs.cloud.google.com/gemini-enterprise-agent-platform/models/model-garden/use-models#deploy_a_partner_model_and_make_prediction_requests) .
