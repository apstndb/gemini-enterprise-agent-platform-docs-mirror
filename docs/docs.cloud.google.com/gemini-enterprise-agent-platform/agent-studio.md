Agent Studio provides a centralized, collaborative workspace for discovering models, refining system instructions, and optimizing prompts. By integrating built-in experiences with the broader Google Cloud ecosystem, Agent Studio helps you transform innovative ideas into production-ready generative AI solutions.

## Key capabilities

  - **[Interactive canvas view](https://console.cloud.google.com/agent-platform/studio/agent-designer/edit) :** A dual-pane workspace that lets you preview and edit code for the artifacts you are developing, ensuring a seamless transition from visual design to implementation.
  - **[Prompt refinement and comparison](https://console.cloud.google.com/agent-platform/studio/multimodal) :** Use built-in natural language tools to automatically generate system instructions and perform side-by-side comparisons to test how different instructions and models affect agent behavior.
  - **[Efficient developer workflow](https://docs.cloud.google.com/gemini-enterprise-agent-platform/agent-studio/agent-studio-capabilities#studio_discovery_and_development) :** Navigate large prompts with a summarized history **minimap** . Use **slash commands** or ask for help using Google Cloud assistance to manage tools, handle conversation context, and generate sample code quickly.
  - **Grounding and RAG:** Enhance response accuracy by grounding agents in real-time data from **[Google Search](https://docs.cloud.google.com/gemini-enterprise-agent-platform/models/grounding/grounding-with-google-search)** and **[Google Maps](https://docs.cloud.google.com/gemini-enterprise-agent-platform/models/grounding/grounding-with-google-maps)** , or integrate your own enterprise data using **[RAG Engine](https://docs.cloud.google.com/gemini-enterprise-agent-platform/build/rag-engine/rag-overview)** , **[Agent Search](https://docs.cloud.google.com/gemini-enterprise-agent-platform/models/grounding/grounding-with-vertex-ai-search)** , or **[Elasticsearch](https://docs.cloud.google.com/gemini-enterprise-agent-platform/models/grounding/grounding-with-elasticsearch)** .
  - **Granular control and settings:** Fine-tune model performance with adjustable **[thinking levels](https://docs.cloud.google.com/gemini-enterprise-agent-platform/models/thinking)** , **[structured output](https://docs.cloud.google.com/gemini-enterprise-agent-platform/models/capabilities/control-generated-output)** requirements, and customizable **[safety filters](https://docs.cloud.google.com/gemini-enterprise-agent-platform/models/capabilities/configure-safety-filters)** .

Agent Studio offers [advanced model access](https://docs.cloud.google.com/gemini-enterprise-agent-platform/models/google-models) . Build with Google's most capable models, including **Gemini 3.1** for reasoning, **Nano Banana 2** for low-latency image generation, and **Veo 3.1** for high-quality video generation.

Experience seamless integration with Agent Studio's cloud-native integration with the broader Google Cloud ecosystem, bringing you access to APIs and deployments.
