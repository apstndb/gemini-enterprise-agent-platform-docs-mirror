Learn how to get help for agents in Agent Platform, including Google support packages, community discussions, bug reporting, and direct team support.

## Get a Google support package

Google Cloud offers different support packages to meet different needs, such as 24/7 coverage, phone support, and access to a technical support manager. For more information, see [Google Cloud Support](https://docs.cloud.google.com/support) .

## Google Developer Program forums

For community-based support, discussions, and to connect with other users and Google staff, your primary destination is [discuss.google.dev](https://discuss.google.dev/tags/c/google-cloud/14/vertex-ai-agent-builder) . This is the official forum for:

  - **Asking questions.** Get peer-to-peer support for your technical challenges.
  - **Sharing knowledge.** Contribute your expertise and help other members of the community.
  - **Engaging with Google staff.** Discuss with Google Cloud team members and find best practices and tutorials from Google staff.

## File bugs or feature requests

[Send feedback](https://issuetracker.google.com/issues/new?component=1130925&template=0) about our products to the Agent Platform team.

## Get support from the Gemini Enterprise Agent Platform team

To get support from the Gemini Enterprise Agent Platform team, you can do the following:

  - Find the list of all open issues from the [Agent Platform SDK GitHub issues page](https://github.com/googleapis/python-aiplatform/issues) , and search for "reasoning engine" or "langchain". See [GitHub documentation for filtering and searching issues](https://docs.github.com/en/issues/tracking-your-work-with-issues/using-issues/filtering-and-searching-issues-and-pull-requests) for more information.

  - For new issues, [file a new bug report](https://github.com/googleapis/python-aiplatform/issues/new?template=bug_report.md) and include `[Reasoning Engine]` in the issue title.

  - For private preview features, email your issue to `vertex-ai-agents-preview-support@google.com` .

  - For questions and feedback about Managed Agents API on Agent Platform, email the public discussion group at `managed-agents-in-agent-platform@google.com` .
