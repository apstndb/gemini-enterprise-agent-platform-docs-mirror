A TensorboardTimeSeries data point.

Fields

`wallTime` ` string ( Timestamp  ` format)

Wall clock timestamp when this data point is generated by the end user.

Uses RFC 3339, where generated output will always be Z-normalized and use 0, 3, 6 or 9 fractional digits. Offsets other than "Z" are also accepted. Examples: `"2014-10-02T15:01:23Z"` , `"2014-10-02T15:01:23.045123456Z"` or `"2014-10-02T15:01:23+05:30"` .

`step` `string ( int64 format)`

Step index of this data point within the run.

`value` `Union type`

Value of this time series data point. `value` can be only one of the following:

`scalar` ` object ( Scalar  ` )

A scalar value.

`tensor` ` object ( TensorboardTensor  ` )

A tensor value.

`blobs` ` object ( TensorboardBlobSequence  ` )

A blob sequence value.

## TensorboardTensor

One point viewable on a tensor metric plot.

Fields

`value` `string ( bytes format)`

Required. Serialized form of <https://github.com/tensorflow/tensorflow/blob/master/tensorflow/core/framework/tensor.proto>

A base64-encoded string.

`versionNumber` `integer`

Optional. version number of TensorProto used to serialize `  value  ` .
